# Student Finals Grader

<img src="https://i.ibb.co/ZZSVXzL/Screenshot-3.jpg" alt="">

A professional grading application for educators.

<p align="center"> <img src="https://img.shields.io/badge/license-Proprietary-blue" alt="License"> <img src="https://img.shields.io/badge/platform-Windows-lightgrey" alt="Platform"> <img src="https://img.shields.io/badge/python-3.8%2B-green" alt="Python"> <img src="https://img.shields.io/badge/version-1.0.0-brightgreen" alt="Version"> </p>

<b>Features</b>:

- Manual and Automatic grading modes.
- Light and Dark themes.
- Adjustable UI sizes.
- Show/Hide grade letters.
- Storage management.
- Export to CSV, TXT, DOCX, PDF.
- Generate class reports and grade distributions.
- Import students from CSV.
- Auto-incrementing student numbers.
- Student list with grid view.
